1.External Style Sheet
....CSS properties are defined in a seperate file
    and save it as .css

   <link href="cssfilename " rel="stylesheet"> under <head>

....How to define CSS properties
....selector/tagname{cssproperties1:value;
             csspropertie2:value;...}
...To verify the ouput of CSS properties, open the 
       html file in the browser
